About

CaRP - Captcha as a gRaphical Password is a new graphical password scheme which aims at providing dynamic and intuitive password scheme. We have used the MNIST data set to create captcha images. Add to that the functionality of selecting an image at a row-column position in the image created, and some mathematical function on the number at that position to create a password
